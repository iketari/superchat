Тестирование Frontend

Что такое тестирование

Тестирование — процесс испытания продукта, имеющий две цели:

  • продемонстрировать разработчикам и заказчикам, что программа соответствует требованиям
  • выявить ситуации, в которых поведение программы является неправильным, нежелательным или не соответствующим спецификации

Код для людей

Код, который мы пишем, он не для компьютеров, он для людей. Хотелось бы, чтобы он использовался и развивался, без полного переписывания при первой же возможности. Для этого код должен быть

  • тестируемым
  • поддерживаемым
  • читаемым
